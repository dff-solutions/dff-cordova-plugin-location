# demo app for the location plugin (module) - ionic v1 & angularJS

## Requirements
- java
- android sdk
- cordova cli
- ionic cli

## Supported platforms

- Android 

## Plugin@latest

- Android: 8.0.1

# Installation

`npm i`

# Build

`ionic build android`

# Run 
NB: if you want to run the app, you can just skip the build step and enter -->

`ionic run android`
